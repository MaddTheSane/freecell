Aide de Freecell

Préférences

Il y a trois options dans les préférences de Freecell pouvant faciliter le jeu.

Déplacement automatique des cartes vers les piles.
Lorsque cette option est activée, les cartes qui ne sont plus utiles pour le jeu seront si possible déplacées vers les piles. Les as sont toujours empilés et les autres cartes sont empilées lorsqu'elles ne sont plus d'aucune utilité pour la suite du jeu.
Permettre le déplacement de plusieurs cartes
Cette option permet le déplacement de plusieurs cartes d'une colonne à une autre en deux clics seulement. Ceci déplace le même nombre de cartes que si le déplacement était effectué manuellement, mais vous épargne du travail.
