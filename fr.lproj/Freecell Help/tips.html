Aide de Freecell

Je perds constamment!

Apprendre à jouer Freecell est facile mais gagner peut être difficle. Voici quelques trucs pour vous aider.

Utilisez vos cases libres (en haut à gauche) le moins possible
Si vous remplissez toutes vos cases libres tôt dans la partie, vous n'irez probablement pas loin. Essayez d'utiliser une ou deux cases pour commencer. Cherchez à effectuer des déplacements pouvant libérer ces cases.
Empilez les as le plus tôt possible
Sans les as sur les piles, vous ne pouvez pas progresser. Essayez d'en avoir un sur chaque pile le plus tôt possible.
N'emprisonnez pas les cartes de valeur basse.
Il est important de ne pas créer de longues séries sur des colonnes ayant des cartes basses plus haut dans la colonne. Si vous ne pouvez pas atteindre vos deux, trois et quatre, il sera impossible de libérer le jeu de cartes vers les piles. Essayez d'éviter de créer des séries sur une carte inférieure au huit.
Utilisez les cases libres pour déplacer plusieurs cartes
Vous pouvez déplacer plus d'une carte ordonnée d'une colonne à une autre en utilisant les cases vides (haut à gauche). Si vous avez une case libre, vous pouvez en déplacer deux; si vous avez deux cases libres, vous pouvez en déplacer trois; en ainsi de suite.
Utilisez les colonnes vides pour déplacer encore plus de cartes.
Pour chaque colonne libre, vous pouvez déplacer deux fois plus de cartes. Ceci devient particulièrement utile lorsque vous avez deux cases libres et une colonne avec une carte. Avec deux cases libres on peut déplacer trois cartes mais si vous videz la colonne (en mettant la carte dans autre case libre) vous pouvez déplacer quatre cartes.
